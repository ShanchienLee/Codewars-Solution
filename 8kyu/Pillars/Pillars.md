# 題目要求

1. 道路上有柱子，柱子間距皆相同，每根柱子寬度也皆相同。

    a. 柱子數 ≥ 1

    b. 柱子間距單位為公尺

    c. 柱子寬度單位為公分

2. 求第一根柱子至最後一根柱子的距離 (不計入柱子寬度)。

---------

# 解題思路

1. 先建立邏輯計算式

    > 柱子數：num_pill
    >
    > 柱子間距：dist
    >
    > 柱子寬度：width
    >
    > (num_pill - 1) * dist * 100 + (num_pill - 2) * width

    * 注意單位不同

2. 區分兩個情況，一為柱子數 ≥ 2 ， 另一為柱子數 = 1 。

3. 以 if 判斷式建立上述兩種情況。
    >
    >       if num_pill >= 2:
    >           return((num_pill - 1) * dist * 100 + (num_pill - 2) * width)
    >       elif num_pill == 1:
    >           return 0
