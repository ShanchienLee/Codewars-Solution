# 題目要求

1. 故事概要：

    * Bob 是一位公車司機，大家都喜歡搭他的公車，請建立一個函式計算他是否能容納足夠的乘客。
   
2. 任務要求：
   
   * 給定三個參數：

     1. cap 是除司機外，公車能容納的人數
    
     2. on 是除司機外，已搭載的人數
    
     3. wait 是除司機外，尚在等待的人數

    * 如果位置充足，則回傳 0 ；如果位置不夠，則回傳不夠多少位置。

-----------------

# 解題思路

1. 先設定變數。

>       available = cap - on - wait

2. 利用 if 判斷式，如果 available 大於等於 0 ，則回傳 0 ；若否，則回傳 available 的負值。

>       if available >= 0:
>           return 0
>       else:
>            return -available
