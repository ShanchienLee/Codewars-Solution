# 題目要求

1. 題目給定一組由字串及數字組成的整數序列。

2. 回傳該序列為數字的總和。

-----------------

# 解題思路

1. 我們會需要先把題目提供的序列轉換成整數，然後再做加總。

2. 先設定一變數 array ， 作為將題目提供之序列內容轉換為整數序列的容器。

>       array = list(map(int, arr))

   * 這裡用到 map ， 可以將 list 中的元素做轉換，第一個參數是要轉換成型態，第二個參數是要儲存的形式。 

3. 設定一變數 result 作為 序列加總後的容器。

>       result = sum(array)

4. 回傳 result 。
